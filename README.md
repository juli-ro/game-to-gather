
# Spielabend-Organizer - game-to-gather

Webseite: https://gametogather.de/

Eine Webanwendung zur Organisation von Spieleabenden mit Freunden.  
Die Anwendung ermöglicht es, Termine zu planen, Spiele auszuwählen, Mitbringsel wie Snacks zu koordinieren und Teilnehmer abzustimmen – alles an einem Ort.

## Technologien
- **Frontend:** Angular  
- **Backend:** .NET REST API  
- **Datenbank:** MySQL  

## Projektziel
Das Projekt wurde ursprünglich als private Plattform entwickelt, um Spieleabende einfacher zu planen und den Überblick zu behalten.  
Ziel war es, eine zentrale Anwendung zu schaffen, über die man festlegen kann:
- welches Spiel oder welche Spiele gespielt werden,  
- wer welche Spiele oder Snacks mitbringt,  
- und an welchem Termin der Spieleabend stattfindet .  

## Aktueller Stand
